# Clean built vignettes

**\[deprecated\]**

`clean_vignettes()` is deprecated because
[`build_vignettes()`](https://devtools.r-lib.org/dev/reference/build_vignettes.md)
is deprecated.

## Usage

``` r
clean_vignettes(pkg = ".")
```

## Arguments

- pkg:

  The package to use, can be a file path to the package or a package
  object. See
  [`as.package()`](https://devtools.r-lib.org/dev/reference/as.package.md)
  for more information.
