# devtools: Tools to Make Developing R Packages Easier

Collection of package development tools.

## Package options

Devtools uses the following
[`options()`](https://rdrr.io/r/base/options.html) to configure
behaviour:

- `devtools.install.args`: a string giving extra arguments passed to
  `R CMD install` by
  [`install()`](https://devtools.r-lib.org/dev/reference/install.md).

- `devtools.path`: Deprecated. Path used by the now-deprecated
  [`dev_mode()`](https://devtools.r-lib.org/dev/reference/dev_mode.md)
  function.
